<br>

### **주제 선정 배경**
- 코로나19 상황에서 **온라인 쇼핑몰 접속 체류 시간** 증가
- 온라인 쇼핑몰에서 의류 선택을 위한 **고민하는 시간** 증가 <br>

  ➡️ 개인에 맞춘 의류 추천 시스템을 통해 의류 선택에 소모하는 **시간**과 **에너지** 감소

<br>

### **주요 기능**
- 소비자가 입고 있는 옷과 비슷한 스타일 추천
- 초 개인화를 반영한 퍼스널 컬러에 맞는 스타일 추천
- 개인의 퍼스널 컬러 진단하기
